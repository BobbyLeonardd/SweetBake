# Contributing to SweetBake

Terima kasih udah tertarik buat kontribusi ke proyek SweetBake! 

## Cara Kontribusi

1. Fork repository ini
2. Buat branch baru (`git checkout -b feature/AmazingFeature`)
3. Commit perubahan kamu (`git commit -m 'Add some AmazingFeature'`)
4. Push ke branch (`git push origin feature/AmazingFeature`)
5. Buat Pull Request

## Coding Standards

### Flutter/Dart
- Ikutin [Effective Dart](https://dart.dev/guides/language/effective-dart)
- Pake `flutter analyze` sebelum commit
- Format code dengan `dart format .`

### PHP
- Pake PSR-12 coding standard
- Selalu pake prepared statements (PDO)
- Validasi semua input dari user

### Commit Messages
- Pake present tense ("Add feature" bukan "Added feature")
- Pake imperative mood ("Move cursor to..." bukan "Moves cursor to...")
- Limit baris pertama ke 72 karakter atau kurang

## Reporting Bugs

Kalo nemu bug, buat issue dengan info:
- Deskripsi bug yang jelas
- Steps to reproduce
- Expected behavior
- Screenshots (kalo ada)
- Environment (OS, Flutter version, dll)

## Feature Requests

Punya ide fitur baru? Buat issue dengan label "enhancement" dan jelasin:
- Kenapa fitur ini penting
- Gimana cara kerjanya
- Contoh use case

## Questions?

Kalo ada pertanyaan, bisa buat issue dengan label "question" atau hubungi maintainer.

Terima kasih! 🙏
